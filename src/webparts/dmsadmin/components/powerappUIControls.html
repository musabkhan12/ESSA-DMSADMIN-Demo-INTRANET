Sypherix Power Apps UI Control Reference

This document explains all major Power Apps UI controls, their deep properties, when and why to use them, and how to bind them in Gallery, Forms, and App context.

1. Label

Used to display static or dynamic text.

Key Properties

  • Text: What to display. Example: ThisItem.Title
  • Color, Size, Visible, Fill: Formatting and conditional styling
Gallery binding:
Text = ThisItem.Title
Form binding:
Default = ThisItem.Title

2. Text Input

Used to capture user text input.

Key Properties

  • Default: Initial value (e.g., ThisItem.Title)
  • Text: Current user-entered value
  • HintText: Placeholder
  • Mode: SingleLine, MultiLine, Password
Save value:
Patch(MyList, ThisItem, { Title: TextInput1.Text })

3. Check Box

Represents true/false values.

Key Properties

  • Default: Initial checked state (ThisItem.IsActive)
  • Value: True if checked, false if not
  • Text: Label next to box
  • OnCheck / OnUncheck: Actions on change
Gallery binding:
Default = ThisItem.Completed
Save:
Patch(MyList, ThisItem, { Completed: Checkbox1.Value })

4. Toggle

Alternative for Yes/No values with slider.

Key Properties

  • Default: Initial state
  • Value: Current state

5. Dropdown

Single-select dropdown.

Key Properties

  • Items: Data source (e.g., Choices(MyList.Status))
  • Default: Initial option
  • Selected.Value: Current selected option

6. ComboBox

Advanced dropdown for single or multiple selection.

Key Properties

  • Items: Data source (Users, Choices, Collections)
  • DisplayFields: Fields shown in UI (e.g., ["DisplayName","Mail"])
  • SearchFields: Fields searched when typing
  • DefaultSelectedItems: Preloaded values (table required)
  • Selected: Single selected record
  • SelectedItems: Table of records (multi select)
  • AllowMultipleSelection: true/false
People picker binding:
Items = Office365Users.SearchUser({SearchTerm: ComboBox1.SearchText})
DisplayFields = ["DisplayName","Mail"]
DefaultSelectedItems = ThisItem.AssignedTo

7. Radio Button

Choose one option from multiple.

Key Properties

  • Items: Options (["High","Medium","Low"])
  • Default: Initial selection
  • Selected.Value: Current option

8. Slider

Pick numeric values.

Key Properties

  • Min, Max: Range
  • Default: Start value
  • Value: Current value

9. Rating

Star-based rating input.

Key Properties

  • Default: Initial rating (convert string to number)
  • Value: Current stars selected
Binding:
Default = Value(Substitute(ThisItem.'User ratings', ",", ""))

10. Date Picker

Pick a date.

Key Properties

  • DefaultDate: Start date (ThisItem.DueDate)
  • SelectedDate: Chosen date

11. Gallery

Repeats controls for multiple records.

Key Properties

  • Items: Data source (Filter(MyList, Status="Active"))
  • ThisItem: Current record
Label.Text = ThisItem.Title
Checkbox.Default = ThisItem.Completed

12. Forms (Edit/Display)

Used for record create, edit, view.

Key Properties

  • DataSource: The list/table
  • Item: Record to edit (Gallery1.Selected)
  • Default: Field default (from ThisItem)
  • Update: What value to save (e.g., TextInput1.Text)
